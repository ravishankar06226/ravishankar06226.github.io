Projects

RAG Based Chat Bot | POC

  • Implemented a Retrieval Augmented Generation pipeline, where user can chat and asks questions on the context of multiple uploaded PDFs.
  • We have used langchain’s PyPDFLoader as documents loader, RecursiveCharacterTextSplitter as text splitter, chromaDB as vector database, semantic similarity search strategy for retriever and Gemini’s Generative AI model as LLM for generation for pipeline. Also We have used rolling window of 2 for retaining the memory of chatbot.
  • We have used also tried the same pipeline with local LLM deployed using Ollama docker image(Llamma-3 model) along with HuggingFace’s BGE-small-model’s embeddings for vector DB embedding.
